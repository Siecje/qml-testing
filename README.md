# Example QML Application

To demonstrate how to test and package a PyQt5 QML Application.

## Development Instructions

## Linux

```shell
python3 -m venv venv
source venv/bin/activate
pip install PyQt5
python main.py
```

## Windows

```shell
python3 -m venv venv
call venv\Scripts\activate.bat
pip install PyQt5
python main.py
```

Then enter 'user' as the password.
